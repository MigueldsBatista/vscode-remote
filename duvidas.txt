**Perguntas sobre Exames de Imagem (Raio X, Ultrassom, Ecocardiograma, Eletrocardiograma, Endoscopia):**

**1. O que é um Raio X e por que é feito?**
   - Um Raio X é um exame de imagem que utiliza radiação para produzir imagens de estruturas internas do corpo, como ossos e órgãos. É frequentemente usado para diagnosticar fraturas, infecções e outras condições médicas.

**2. Como funciona um Ultrassom?**
   - O Ultrassom usa ondas sonoras de alta frequência para criar imagens dos órgãos internos. É frequentemente usado para monitorar o desenvolvimento fetal durante a gravidez e avaliar órgãos como o fígado e os rins.

**3. O que é um Ecocardiograma?**
   - Um Ecocardiograma é um exame que usa ultrassom para criar imagens do coração. É usado para avaliar a estrutura e o funcionamento do coração, ajudando no diagnóstico de doenças cardíacas.

**4. Quando é necessário fazer um Eletrocardiograma?**
   - Um Eletrocardiograma mede a atividade elétrica do coração e é usado para diagnosticar problemas cardíacos, como arritmias. É frequentemente solicitado quando há sintomas cardíacos.

**5. O que envolve um procedimento de Endoscopia?**
   - A Endoscopia é um procedimento em que um tubo flexível com uma câmera é usado para examinar o interior do trato gastrointestinal. É útil no diagnóstico de problemas digestivos.
